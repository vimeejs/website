## With an editor UI

vimee provides several editor plugins for different use cases:

| Package | Description |
|---------|-------------|
| `@vimee/shiki-editor` | Syntax-highlighted editor powered by Shiki |
| `@vimee/plugin-textarea` | Simple textarea-based editor |
| `@vimee/plugin-monaco` | Monaco Editor integration |
| `@vimee/plugin-codemirror` | CodeMirror integration |

Install the one that fits your needs:

```bash
npm install @vimee/core @vimee/plugin-monaco
```

## Requirements

- Node.js 18+
- TypeScript 5.0+ (recommended)
