## エディタープラグインと一緒に

vimee はさまざまなユースケースに対応するエディタープラグインを提供しています：

| パッケージ | 説明 |
|---------|-------------|
| `@vimee/plugin-textarea` | 任意の HTML textarea に Vim キーバインドを付与 |
| `@vimee/plugin-monaco` | Monaco Editor 統合 |
| `@vimee/plugin-codemirror` | CodeMirror 6 統合 |

ニーズに合ったものをインストールしてください：

<Tabs labels={["npm", "yarn", "pnpm", "bun"]}>
  <div class="tab-panel">
    ```bash
    npm install @vimee/core @vimee/plugin-textarea
    ```
  </div>
  <div class="tab-panel">
    ```bash
    yarn add @vimee/core @vimee/plugin-textarea
    ```
  </div>
  <div class="tab-panel">
    ```bash
    pnpm add @vimee/core @vimee/plugin-textarea
    ```
  </div>
  <div class="tab-panel">
    ```bash
    bun add @vimee/core @vimee/plugin-textarea
    ```
  </div>
</Tabs>
