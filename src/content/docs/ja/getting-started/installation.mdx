## エディター UI と一緒に

vimee はさまざまなユースケースに対応するエディタープラグインを提供しています：

| パッケージ | 説明 |
|---------|-------------|
| `@vimee/shiki-editor` | Shiki を使用した構文ハイライトエディター |
| `@vimee/plugin-textarea` | シンプルな textarea ベースのエディター |
| `@vimee/plugin-monaco` | Monaco Editor 統合 |
| `@vimee/plugin-codemirror` | CodeMirror 統合 |

ニーズに合ったものをインストールしてください：

```bash
npm install @vimee/core @vimee/plugin-monaco
```

## 要件

- Node.js 18+
- TypeScript 5.0+（推奨）
